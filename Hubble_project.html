Estimating the Hubble Constant via Gravitational-Wave Dark Sirens

Overview

Precise determination of the Hubble constant (H₀) remains a central puzzle in cosmology, with early- and late-Universe methods in tension. This project uses gravitational-wave observations of binary black-hole mergers—so-called “dark sirens”—to measure H₀ independently of electromagnetic data.

Aims

  • Validate a statistical framework for associating each event with its host galaxy.
  • Recover frequentist point estimates of H₀ via combined likelihood maximization.
  • Quantify uncertainties using Monte Carlo resampling and Bayesian posterior construction.

Methods

  • Generate distance measurements (30% Gaussian uncertainty) and sky localizations (von Mises–Fisher, 5° spread).
  • Draw candidate hosts from EAGLE simulation data, filtered by stellar mass.
  • Compute joint angular & distance likelihoods for N = 100 events.
  • Maximize summed likelihood for frequentist H₀ estimates; resample for confidence intervals.
  • Build Bayesian posterior with uniform prior on H₀ ∈ [40, 100] km⋅s⁻¹⋅Mpc⁻¹.

Results

Across four independent runs (100 events each), recovered H₀ values cluster 68–72 km⋅s⁻¹⋅Mpc⁻¹, bracketing the true input of 67 km⋅s⁻¹⋅Mpc⁻¹. The Bayesian posterior yields H₀ = 72.0−19.8+17.9 km⋅s⁻¹⋅Mpc⁻¹ (68% credible interval). Spline smoothing reduces Monte Carlo noise effectively.

Conclusions & Future Work

Gravitational-wave dark sirens can provide meaningful constraints on the expansion rate, independent of electromagnetic counterparts. Scaling to larger samples and deeper galaxy catalogs will be crucial to resolving the Hubble tension in upcoming gravitational-wave surveys.
